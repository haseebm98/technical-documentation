Introduction

HTML, standing for "HyperText Markup Language", is a markup language used to design web pages. It is a combination of Hypertext and Markup language. Hypertext is the element that defines links between web pages, while a markup language is used to define the text document inside the web page. Markup languages are human-readable, and uses tags to manipulate the text and data within.

HTML was created by Tim Berners-Lee in 1991, but wasn't released till two years later. The image below shows the years in which each version of HTML was released, up until the newest version: HTML5.

Basics

HTML uses tags to manipulate information. In this section, we will learn the most important tags every HTML page must include. Some tags are self closing, so don't need a tag to declare its end. However, most tags need a closing tag, normally the same as the opening tag, but with a / e.g. <p> and </p>. Below, we list the basic elements in every HTML page.

  • <!DOCTYPE html>: This goes at the top of the page, before any other tag, and declares to the web that this page is a HTML type document.
  • <html>: Every HTML page must be enclosed between these tags. It begins with this and ends with its relevant closing tag.
  • <head>: This tag comes after the HTML tag, and contains all the header information about the web page, like the title and other miscellaneous information. It must be closed with its closing tag.
  • <title>: This goes inside the head tag, and declares the title that will be visible on the tab of the internet browser. This must be closed with its closing tag.
  • <body>: This is arguably the most important tag. It contains the whole visible body of the website. Once again, it is closed with its relevant tag. Everything enclosed within these tags will be visible to the viewer.
Heading

An HTML heading tag is used to define headings of a page. There are six levels of headings, from <h1> through to <h6>. <h1> is the highest level of headings, and <h6> is the lowest. All heading tags must be closed.

  • <h1>: used for main headings in a page
  • <h2>: used for subheadings
  • <h3> - <h6>: if there are other lower level headings required inside subheadings, these are normally used

The sizes of each heading varies with the browser, but they will always be in size order, i.e heading 1 will always be larger than 2, 3, ... down to 6. Their individual sizes can be manipulated further through the use of CSS.

Paragraphs

In HTML, the simplest way to write paragraphs is through the use of <p> tags. Again, these require closing tags. When using these, any text enclosed between them will display as one continuous paragraph, so no line break or double spacing will be visible. To write a new line, it is common practice to simply use new paragraph tags.

The <br> tag is a way of inserting line breaks both inside and outside paragraph tags. Inside the paragraph tags, it acts as a way of separating parts of the paragraph onto a new line. This tag has no closing tag.

Inside the <p> tag, we can declare attributes to change the layout. For example, we can use <p align = "center"> to align all text in that paragraph to the center of the page. The same can be done with left and right alignment.

Lastly, there exists another tag, called the <pre> tag. Any text enclosed within these tags will display any extra spacing or line breaks. Unlike the paragraph tag, it allows the use of extra spacing and line breaks without the use of another tag.

Text Formatting

Below, we will list some of the possible text formatting that can be done without the use of CSS.

  • <strong>: Makes text bold/strong
  • <b>: Makes text bold
  • <small>: Makes text small
  • <mark>: Highlights text in yellow
  • <i>: Makes text italic
  • <em>: Makes text emphasised (italic)
  • <sup>: Puts text in superscript
  • <sub>: Puts text in subscript
  • <del>: Strikes through text
  • <ins>: Underlines text
